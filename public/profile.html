Job Queue Profile

This is the profile for the job queue.

Link relations

These are in addition to those in the IANA link registry.

  • template: Following a link with this relation will give you a template object used to create a new job.

The job Object

Some notes on the job object:

Creation

To create a job, send a filled-in template via a POST request to a link with the index relation.

Links

The links attribute isn't part of the data representation of the job object, it simply contains a list of links. This will be a JSON array with objects with two keys: rel and href.
